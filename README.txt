A simple feedback form plugin for WordPress.

== Description ==
Jovi Feedback form plugin is a powerful, easy-to-use WordPress plugin designed for collecting feedback, survey data from users and analyze it. It is a free and open source software which you can add to any page or post of your website simply by adding just a shortcode. It provides the submitted form-data right into your admin area without requiring any setup. 

It has features like unread notification, new-read message, delete message - all the features you need for a simple feedback or survey form for your website. 

== Installation ==
1. Upload the plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the \'Plugins\' menu in WordPress.
3. Use the shortcode `[jovi_feedback_form]` in your page/post.

== Frequently Asked Questions ==
How to add Form to the site? =
Use the shortcode `[jovi_feedback_form]` in any post, page, or widget area where you want the feedback form to appear.

Is this plugin free?
Yes, it is completely f free and open source software.
